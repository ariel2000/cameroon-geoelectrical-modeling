# Численное моделирование геоэлектрических методов для разведки тропических полезных ископаемых
## Синтетические отклики постоянного тока, индуцированной поляризации и электромагнитной разведки во временной области для типичных геологических моделей в Камеруне

### Описание проекта

Этот репозиторий содержит численные модели, предназначенные для исследования отклика геоэлектрических методов в геологических условиях, репрезентативных для целей разведки полезных ископаемых в Камеруне и Центральной Африке.

Исследование сосредоточено на эффективности трех взаимодополняющих геофизических методов:

- Резистивное сопротивление постоянного тока (DC)
- Индуцированная поляризация (IP)
- Электромагнетизм во временной области (TDEM)

Цель состоит в оценке того, как проводящие приповерхностные слои, характерные для профилей тропического выветривания, влияют на обнаруживаемость и оценку параметров более глубоких сульфидных минерализаций.

Поскольку доступ к полевым геофизическим данным из Камеруна ограничен, исследование основано на **синтетическом моделировании**, которое обеспечивает контролируемую основу для анализа чувствительности параметров, пределов разрешения и методологических ограничений.


Все моделирования выполнены с использованием геофизической платформы с открытым исходным кодом **SimPEG**.

Прямые задачи решаются численно с использованием **конечно-элементной дискретизации**.

--

# Геологический контекст

Тропические регионы, такие как Камерун, характеризуются интенсивными процессами химического выветривания, которые приводят к образованию толстых профилей реголита. Эти профили обычно включают:

- Латеритные горизонты, богатые оксидами железа
- Сапролитовые зоны, образованные из измененных коренных пород
- Относительно резистивный свежий фундамент

Такие профили выветривания существенно влияют на геофизические отклики, поскольку приповерхностные слои часто являются проводящими.

Эти проводящие горизонты влияют на:

- распределение тока в методах постоянного тока
- поляризационные эффекты в методе IP
- электромагнитную диффузию в методе TDEM

Понимание этих эффектов имеет важное значение для интерпретации геофизических данных в условиях тропической разведки.

# Концептуальная геологическая модель

Синтетическая модель представляет собой упрощенный, но геологически реалистичный профиль тропического выветривания.

### Слоистая структура

| Слой | Толщина | Удельное сопротивление |

|------|------|------|

| Латерит | 15 м | 80 Ом·м |

| Сапролит | 30 м | 200 Ом·м |

| Фундамент | полубесконечный | 1000 Ом·м |

### Минерализованная цель

| Параметр | Значение |

|------|------|

| Глубина до вершины | 100 м |

| Ширина | 80 м |

| Высота | 40 м |

| Удельное сопротивление | 10 Ом·м |

| Проводимость | 0,1 С/м |

| Зарядоспособность | высокая |

Проводящая аномалия представляет собой упрощенное **сульфидное минерализованное тело**, типичное для полиметаллических месторождений.

# Численный подход к моделированию

Процесс моделирования включает несколько этапов:

1. Определение геологической концептуальной модели.

2. Параметризация с использованием репрезентативных петрофизических значений.

3. Построение вычислительной сетки.

4. Моделирование геофизических откликов с использованием прямого моделирования.

5. Визуализация синтетических результатов.

Прямое моделирование выполняется с помощью **SimPEG**, который решает уравнения Максвелла и электростатические уравнения с использованием дискретизации методом конечных элементов.

Реализованы три подхода к моделированию:

- **2D моделирование удельного сопротивления постоянного тока**

- **2D моделирование наведенной поляризации**
- **1D моделирование TDEM-слоев**

Эти моделирования генерируют синтетические наборы данных, иллюстрирующие ожидаемый отклик каждого геофизического метода.

# Установка

Рекомендуется использовать Python 3.9 или более позднюю версию.

Установите необходимые библиотеки:
pip install numpy matplotlib simpeg discretize pymatsolver scipy
---

# Воспроизведение результатов

Перейдите в каталог проекта:
cd cameroon-geoelectrical-modeling
Создайте геологическую эталонную модель:
python scripts/plot_reference_model.py

Запустите моделирование удельного сопротивления постоянного тока:
python scripts/dc_forward_2d.py

Запустите моделирование наведенной поляризации:
python scripts/ip_forward_2d.py

Запустите моделирование TDEM:
python scripts/tdem_forward_1d.py

Все данные будут автоматически сохранены в каталоге **outputs**.

--

# Сгенерированные синтетические данные

В результате моделирования были получены следующие данные:

- Модель геологического удельного сопротивления
- Псевдосечение кажущегося удельного сопротивления постоянного тока
- Псевдосечение заряжаемости IP
- Кривая затухания переходного процесса TDEM

Эти результаты иллюстрируют взаимодополняющую чувствительность трех геофизических методов.
