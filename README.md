# cameroon-geoelectrical-modeling
Synthetic modeling of DC, IP and TDEM responses for typical geological models of Cameroon deposits.
# Проект геоэлектрического моделирования в Камеруне

## Обзор

В рамках этого проекта исследуется возможность применения геоэлектрических методов (сопротивление постоянного тока, индуцированная поляризация и TDEM) для оценки параметров геологических моделей, представляющих собой месторождения полезных ископаемых в Камеруне.

Исследование основано на **синтетических данных, полученных с помощью численного моделирования**. Главная цель — проанализировать, как проводящие приповерхностные слои (латерит и сапролит) влияют на возможность обнаружения глубинной сульфидной минерализации.

Численное моделирование выполняется с использованием геофизической платформы с открытым исходным кодом **SimPEG**.

Прямые задачи решаются с использованием **методов конечных элементов**.

## Геологическая концептуальная модель

Синтетическая модель представляет собой типичный профиль тропического выветривания, распространенный в Центральной Африке.

Слои:

| Слой | Толщина | Сопротивление |

|------|------|------|

| Латерит | 15 м | 80 Ом·м |

Сапролит | 30 м | 200 Ом·м |

Фундамент | полубесконечный | 1000 Ом·м |

Целевое тело:

| Параметр | Значение |

|------|------|

| Глубина до вершины | 100 м |

| Ширина | 80 м |

| Высота | 40 м |

| Удельное сопротивление | 10 Ом·м |

| Проводимость | 0,1 С/м |

| Зарядимость | высокая |

Аномалия представляет собой **сульфидное минерализованное тело**.
