# Обзор литературы

## Геологический контекст Камеруна

Камерун расположен в пределах Центральноафриканского панафриканского пояса.
Геологическая структура представлена ​​преимущественно докембрийскими породами фундамента,
крупными зонами сдвига и вулканическими структурами, связанными с Камерунской
вулканической линией.

В этих геологических условиях встречаются различные типы минерализации,
включая месторождения золота, железа и сульфидов цветных металлов.

Понимание геологической структуры необходимо для построения
реалистичных синтетических геоэлектрических моделей.
